Aufgaben

  1. Fügen Sie der App zwei Komponenten hinzu:
    • Eine ActiveUser-Komponente und eine UserData-Komponente
    • ActiveUser sollte einen Benutzernamen (h2) und ein Alter (h3) ausgeben
    • UserData sollte zwei Eingabefelder ausgeben => für Name und Alter
    • Optional: Fügen Sie ein Styling Ihrer Wahl hinzu
  2. Geben Sie beide Komponenten nebeneinander in Ihrer Haupt-App-Vorlage aus
  3. Fügen Sie Benutzerdaten hinzu und stellen Sie sicher, dass sie einen Namen und ein Alter enthalten
    • Benutzerdaten sollten in ActiveUser ausgegeben werden
    • Sie sollten über die UserData-Komponente aktualisiert werden
